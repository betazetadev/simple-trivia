# Simple Trivia Game

This is a simple Trivia web game created as an example for a tutorial.

## Technologies Used
- JavaScript
- CSS
- HTML

## How to Run
1. Clone the repository:
  ```sh
   git clone https://github.com/betazetadev/simple-trivia.git
  ```
2. Open `index.html` in your browser.
